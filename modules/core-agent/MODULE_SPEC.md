# Core-Agent 模块规范 (Module Spec)

## 一、 模块定位
本模块负责管理 Claw-Swarm 项目中与 **Agent (智能体) 交互** 相关的底层逻辑、会话状态以及基础行为约束。
它不处理任何具体的系统部署或业务通信，仅负责"守护边界"。

## 二、 对外约束
1. 其他模块的逻辑开发，不得覆盖或绕过本模块定义的守门人 (Gatekeeper) 协议。
2. 任何与 GitHub Issue、反馈流程相关的交互，均由本模块处理。

## 三、 下属功能树 (Features)
| 功能目录 | 状态 | 说明 |
|---|---|---|
| `dual-mode/` | ✅ 完善 | 双轨模式核心逻辑：强制初始化拦截、会话锁定、工作流规范 |
| `model-manager/` | ✅ 完善 | 模型配置诊断与修复、配置层级规则、Skill 支持 |

## 四、 关键配置文件
- `AGENTS.md` — Agent 守门人入口，包含双轨拦截 + 渐进式检索元指令
- `.github/ISSUE_TEMPLATE/app_feedback.md` — 应用模式反馈模板
