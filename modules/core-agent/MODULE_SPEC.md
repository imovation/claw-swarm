# Core-Agent 模块规范 (Module Spec)

## 一、 模块定位
本模块负责管理 Claw-Swarm 项目中与 **Agent (智能体) 交互** 相关的底层逻辑、会话状态以及基础行为约束。
它不处理任何具体的系统部署或业务通信，仅负责“守护边界”。

## 二、 对外约束
1. 其他模块的逻辑开发，不得覆盖或绕过本模块定义的守门人 (Gatekeeper) 协议。
2. 任何与 GitHub Issue、反馈流程相关的交互，均由本模块处理。

## 三、 下属功能树 (Features)
- **`dual-mode/`**：双轨模式核心逻辑。包含模式拦截定义、App 模式与 Dev 模式的工作流闭环约束，以及反馈模板。
